## Demo

Setup:

<https://elixir-lang.org/install.html>

```
$ git clone https://github.com/bdeak4/elixir-demo
$ cd elixir-demo
$ mix deps.get
$ mix ecto.create
$ mix ecto.migrate
```

Pokretanje lokalnog servera:

```
$ mix phx.server
```
